# Car Purchase Prediction (Linear Regression)

This project uses linear regression to predict car purchase amounts based on user features.

## Files
- car_purchasing_lr.ipynb: main model notebook
- export scripts: data processing utilities

## Datasets
Obtained from Kaggle by user (Fathy sahlool )
URL of dataset obtained: https://www.kaggle.com/code/fathyfathysahlool/linear-regression/notebook
